ICT Ops Odoo SMS Marketing Module

A powerful and comprehensive SMS marketing solution for Strathmore University, that enables departments to send bulk SMS campaigns, manage recipients, track delivery status.

Odoo 19.0 SMS Marketing Bulk SMS Africa's Talking

Key Features

Campaign Management

  • Create and manage SMS campaigns with ease
  • Schedule campaigns for future delivery
  • Track campaign status in real-time
  • View detailed statistics and success rates
  • Monitor sent, failed, and pending messages
  • Duplicate existing campaigns for quick setup

Recipient Management

  • Import recipients from CSV files
  • Manually add recipients to campaigns
  • Track individual recipient status
  • View delivery confirmation and error messages
  • Bulk recipient operations

SMS Blacklist

  • Prevent sending SMS to unwanted recipients
  • Maintain a blacklist of phone numbers
  • Automatically prevent sending to blacklisted numbers
  • Add reasons for blacklisting
  • Manage active/inactive blacklist entries
  • Import blacklist from external sources

Gateway Support

  • Africa's Talking - Full integration with AT API
  • Custom API - Connect to any SMS gateway
  • Configure multiple gateways
  • Set default gateway for campaigns
  • Test gateway connections

Analytics & Reporting

  • Real-time campaign statistics
  • Success rate calculations
  • Failed message tracking with error details
  • Delivery confirmation timestamps
  • Export reports for analysis

Installation

  1. Download or clone the module to your Odoo addons directory
  2. Update the app list in Odoo
  3. Install the "ICT Ops Odoo SMS" module
  4. Configure your SMS gateway in Configuration menu
  5. Start creating SMS campaigns!

Configuration

Setting up Africa's Talking

  1. Go to SMS Marketing -> Configuration -> SMS Gateways
  2. Create a new gateway configuration
  3. Select "Africa's Talking" as gateway type
  4. Enter your API Key and Username
  5. Set your Sender ID
  6. Click "Test Connection" to verify

Usage Examples

Creating Your First Campaign

  1. Navigate to SMS Marketing -> Campaigns -> SMS Campaigns
  2. Click "Create" to start a new campaign
  3. Write your message in the Message tab
  4. Import recipients using CSV or add manually
  5. Click "Send Now" or "Schedule" for later delivery

CSV Import Format

Your CSV file should have the following columns:

name,phone_number
                John ,+254712345678
                Jane ,+254723456789
                Bob ,+254734567890
